Part 2: Utilize Github Codespaces

  1. Open the repository in Github and click on the "Code" button, then select "Open with the Codespaces".
  2. Customize the Codespace configuration as needed.
  3. Use the integrated VSCode environment provided by Github Codespaces to continue working on your project.
  4. Create a basic CSS file named styles.css inside the css folder.
  5. Link the style.css file to the index.html
  6. Style the header,navigation bar, main content sction, and footer with basic CSS to make them visually appealing using Flexbox properties.

Part 3: Port Forwarding (NPX)

  1. Install NPX globally if not already installed using the command npm install -g npx.
  2. Use NPX serve to serve your project locally. Run npx serve in the root directory of your project.

Part 4: Tunneling (NGROK)

  1. Utilize WSL for tunneling with ngrok:
  2. Open WSL
  3. Download and install Ngrok from https://ngrok.com
  4. Run Ngrok in the WSL terminal with the command <local port> replacing <local_port> with the port where you local server is running.
  5. Ngrok will provide will provide a public URL that tunnels t your local server.
  6. Commit your changes and push them to the remote repository on Github.
